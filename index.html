快捷键与操作说明
依赖箭头
Alt + 拖拽在信号间创建依赖箭头
双击箭头添加文字标注
点选箭头 + Delete删除箭头
波形编辑
在波形区拖拽设置信号电平（高/低/X/Z）
拖拽跳变沿微调跳变位置
双击总线段编辑总线值
双击信号名重命名信号
全局快捷键
Ctrl + Z撤销
Ctrl + Y重做
Delete删除选中项（信号/箭头/分隔符）
图片识别
Ctrl + V粘贴图片 = 识别波形（黑白无水印图片识别效果最佳）
点击🖼️ 识别图片从本地选择波形图

💡 Mac 用户：Alt = Option (⌥)，Ctrl = Cmd (⌘)
