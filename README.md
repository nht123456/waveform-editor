# Waveform Editor - 在线波形图编辑器

> 告别 WaveDrom JSON 语法，所见即所得画波形图

**在线使用：** https://nht123456.github.io/waveform-editor/

## 为什么做这个？

传统画波形图的方式（如 WaveDrom）需要手写 JSON 代码：

```json
{ "signal": [
  { "name": "clk", "wave": "p......." },
  { "name": "req", "wave": "0.1..0.." },
  { "name": "ack", "wave": "0..1..0." }
]}
```

**痛点：**
- 语法容易写错，调试困难
- 无法直接拖拽调整时序
- 不支持依赖箭头标注（setup/hold time）
- 修改一个跳变沿需要重新数波形字符

**本工具的解决方案：** 直接在画布上拖拽画波形，所有操作即时可见。

## 核心功能

### 1. 所见即所得编辑

| 操作 | 方式 |
|------|------|
| 画高/低电平 | 在波形区直接拖拽 |
| 调整跳变沿 | 拖拽跳变沿位置 |
| 编辑总线值 | 双击总线段输入值 |
| 添加时钟 | 点击 "+ 时钟" 按钮 |
| 重命名信号 | 双击信号名 |

### 2. 依赖箭头标注

- **Alt + 拖拽** 在信号间创建依赖箭头（Mac: Option + 拖拽）
- 支持同信号/跨信号弧线箭头
- 双击箭头添加文字标注（如 "tSU", "tHD"）
- 属性面板可调节曲率、线型（弧线/直线）、方向（单向/双向）

### 3. AI 图片识别（零代码导入波形）

- **Ctrl+V 粘贴截图** → 自动识别为可编辑波形
- 或点击 "🖼️ 识别图片" 按钮上传波形图
- 支持信号名 OCR、总线类型检测、总线段值识别
- 纯前端实现，无需后端服务，数据不上传

> 💡 黑白无水印的波形截图识别效果最佳

### 4. 多格式导出

| 格式 | 说明 |
|------|------|
| PNG | 高清 2x 倍率图片 |
| SVG | 矢量图，可嵌入文档 |
| JSON | 数据格式，可程序化处理 |
| .wfp | 项目文件，支持多 Sheet |
| 独立版 HTML | 单文件包含所有代码和数据，发给别人直接打开 |

### 5. 其他特性

- 多 Sheet 页签管理（类似 Excel）
- 撤销/重做（Ctrl+Z / Ctrl+Y）
- 信号颜色自定义（6 色预设）
- 分隔符（Gap）标注
- 时间轴缩放
- 数据保存在浏览器本地，不丢失

## 与 WaveDrom 对比

| 维度 | WaveDrom | 本工具 |
|------|----------|--------|
| 编辑方式 | 写 JSON 代码 | 鼠标拖拽 |
| 学习成本 | 需记忆语法 | 零学习成本 |
| 依赖箭头 | 有限支持 | 完整支持（弧线/直线/曲率/标注） |
| 图片导入 | 不支持 | AI 识别截图 → 可编辑波形 |
| 导出分享 | 需搭建环境 | 导出独立 HTML，任何浏览器打开 |
| 总线编辑 | 手动写值 | 双击直接编辑 |
| 多页管理 | 不支持 | 多 Sheet 标签页 |
